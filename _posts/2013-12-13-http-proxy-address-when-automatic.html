In corporate environments users are usually connected to internet using HTTP proxy. To ease administration usually automatic configuration is used:



But some programs do not support such configuration so following are the steps to find real proxy address:
  1. Go to address http://wpad.example_domain.com/wpad.dat which should offer you a file for download.
  2. Inspect the file in text editor - search for PROXY statement - it should state your proxy address.
