It was never easy to install Oracle Database on Linux - too many steps to setup users, kernel parameters and all that stuff. Yesterday I tried again and now it seems things are moving forward. To get on the fast lane follow these steps:
  1. Install Oracle Linux. Download ISO image from Oracle and create new virtual machine. Let the VM have at least 16GB of disk space, otherwise you can have space issues.
  2. Download Oracle Database installation files (linux_11gR2_database_1of2.zip and linux_11gR2_database_2of2.zip) to your VM - you can use /tmp and extract them using unzip.
  3. Install package oracle-rdbms-server-11gR2-preinstall (yum install oracle-rdbms-server-11gR2-preinstall). This does the real magic as it preconfigures everything for you.
  4. Start the DB installation (/tmp/database/runInstaller)
