This post is for me so that I don't have to search Google. The example redirects calls to port 8690 to port 3690 where actual server sits. It also saves the configuration so it survives restart.

# iptables -t nat -A PREROUTING -i eth0 -p tcp --dport 8690 -j REDIRECT --to-port 3690

# /sbin/service iptables save

To delete existing rule just replace -A parameter with -D. For example following command deletes forwarding set in previous example:

# iptables -t nat -D PREROUTING -i eth0 -p tcp --dport 8690 -j REDIRECT --to-port 3690
