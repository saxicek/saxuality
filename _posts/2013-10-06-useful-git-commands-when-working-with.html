I am assuming that you already cloned some project using Github web. You should also configure your local repository to have 2 remotes:
  1. origin - this one is created by default and points to your cloned Github project
  2. upstream - this one should point to development branch of the original project. Use following command:
    git remote add upstream https://github.com/midgetspy/Sick-Beard.git

Following command will create branch in which you will be doing your changes. Newly created branch will be used in pull request later:
git checkout -b your_branch_name upstream/development_branch

Following command will push your changes to your Github cloned repository:
git push -u origin your_branch_name

And now you can create pull request using Github web from your branch to the original project.
