Today I am trying to make Fabric work so that I can automate my EC2 deployment. Unfortunately I am working on Windows 7 64-bit which is a bit pain for Python package management. Not everything works as it should. PyCrypto is one example. I should also note that I am using virtualenv but that is not too important for this case. So enough talking, how to make it work?

Download egg from https://tahoe-lafs.org/source/tahoe-lafs/deps/tahoe-lafs-dep-eggs/.
Install it using:
easy_install pycrypto-2.5-py2.7-win-amd64.egg
Simple right? Big thanks goes to tahoe-lafs.org for providing the binaries!
